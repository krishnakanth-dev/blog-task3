# Blog Management System

## Overview

This project is a PHP and MySQL-based Blog Management System developed as part of the ApexPlanet Web Development Internship.

The application allows users to register, log in, and manage blog posts through complete CRUD (Create, Read, Update, Delete) operations.

## Features

### User Authentication

* User Registration
* User Login
* User Logout
* Password Hashing
* Session Management

### Blog Management

* Create Posts
* View Posts
* Edit Posts
* Delete Posts

### Database Integration

* MySQL Database
* phpMyAdmin Management
* Database Connectivity using PHP

## Setup Instructions

1. Install XAMPP.
2. Start Apache and MySQL.
3. Create a database named `blog`.
4. Import or create the required tables.
5. Place the project folder inside:

C:\xampp\htdocs\

6. Open:

http://localhost/blog-project
